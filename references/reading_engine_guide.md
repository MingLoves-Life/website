# 命盘解读引擎指南

基于 skill 原始方法论提炼，并补全健康、贵人、决策择时三个维度。供生成个性化解读内容时使用。

---

## 一、解读风格

- 像朋友聊天，不像老师讲课
- 有主见，敢说"这个格局倾向于X"
- 先看积极面，再温和指出需注意之处
- 用类比翻译术语，禁说"命中注定"、"你一定会"

---

## 二、解读维度（共 8 个）

### 1. 命格总评（命宫主星）

**数据来源**：命宫主星组合 + 身宫位置 + 生年四化落点

**输出内容**：
- 核心判断（一句话概括格局）
- 性格关键词 3-5 个
- 展开说明（星曜如何作用）
- 实际影响（现实生活意味什么）
- 置信度条（初始 65-75%）

**空宫处理**：借对宫主星解读，说明"这个领域更灵活、更受外界影响"。

---

### 2. 十年大运（大限走势）

**数据来源**：大限宫位序列 + 每步大限主星 + 大限四化

**输出内容**：
- 每步大运的核心主题词（如"蛰伏蓄力"、"贵人助力"、"事业巅峰"）
- 运势强度评分（0-100）
- 当前所处大限的详细解读：核心课题、机遇、风险
- 化忌落点对该十年的影响

**当前大限判定**：
- iztro 排盘结果包含每步大限的起止年龄
- 用户当前年龄落在哪步大限区间 → 即为"当前大限"
- 当前大限需详细展开，其余大限只给主题词和评分

**量化规则**（基准 50 分，满分 100）：
- 化禄/化权落本宫或三方 → +15~20
- 化科落本宫 → +10
- 化忌落本宫 → -20
- 化忌落三方 → -10
- 煞星（擎羊/陀罗/火铃）集聚 → -10
- 吉星（左辅/右弼/天魁/天钺）辅佐 → +10
- 主星庙旺 → +5
- 主星落陷 → -5

---

### 3. 流年运势

**数据来源**：流年天干四化 + 流年飞入宫位 + 与大限叠加

**输出内容**：
- 上半年/下半年整体走向判词
- 关键月份标注（化忌月、化禄月）
- 综合评分（星级）

**关键月份推导**：
- 流年化忌星所在的月令地支 → 该月份需谨慎
- 流年化禄星旺月 → 该月份有机遇
- 生年忌 + 流年忌 同宫月份 → 重点标红

---

### 4. 事业与财运（官禄宫 + 财帛宫）

**数据来源**：官禄宫主星 + 财帛宫主星 + 两宫四化 + 大限/流年飞入

**输出内容**：
- 事业评分（星级）+ 一句建议
- 财运评分（星级）+ 财运模式（正财/偏财）
- 最佳方位：根据财帛宫地支对应方位
- 旺财月份：化禄飞入财帛宫的流月

**方位对照**：
| 地支 | 方位 |
|------|------|
| 子 | 正北 |
| 丑、寅 | 东北 |
| 卯 | 正东 |
| 辰、巳 | 东南 |
| 午 | 正南 |
| 未、申 | 西南 |
| 酉 | 正西 |
| 戌、亥 | 西北 |

---

### 5. 姻缘与桃花（夫妻宫）

**数据来源**：夫妻宫主星 + 桃花星（贪狼、廉贞、文曲、天姚）位置 + 夫妻宫四化

**输出内容**：
- 桃花指数（星级）
- 桃花月：桃花星所在宫位对应的月令
- 正缘特征：夫妻宫主星推导（年龄倾向、性格特征）
- 感情方位：夫妻宫地支对应方位
- 一句感情建议

**正缘特征推导规则**：
| 夫妻宫主星 | 伴侣特征倾向 |
|-----------|-------------|
| 紫微 | 有气场、年长、领导型 |
| 天机 | 聪明、多变、年轻 |
| 太阳 | 热情大方、外向 |
| 武曲 | 务实果断、能赚钱 |
| 天同 | 温和体贴、居家型 |
| 廉贞 | 有魅力、感情浓烈 |
| 天府 | 稳重可靠、有安全感 |
| 太阴 | 温柔细腻、内向 |
| 贪狼 | 多才多艺、有吸引力 |
| 巨门 | 口才好、有思想 |
| 天相 | 可靠有原则、稳定 |
| 天梁 | 成熟有智慧、年长 |
| 七杀 | 个性强势、独立 |
| 破军 | 不安分、有冲劲 |

### 6. 健康注意（疾厄宫）【补全】

**数据来源**：疾厄宫主星 + 疾厄宫煞星 + 化忌是否飞入疾厄宫 + 五行偏缺

**输出内容**：
- 注意部位/系统
- 易犯月份
- 调理方向

**星曜对应身体部位**：
| 星曜 | 对应部位/系统 |
|------|-------------|
| 紫微 | 头部、神经系统 |
| 天机 | 肝胆、四肢关节 |
| 太阳 | 眼睛、心脏、头部 |
| 武曲 | 肺、呼吸系统、筋骨 |
| 天同 | 肾、泌尿系统、耳朵 |
| 廉贞 | 心脏、血液 |
| 天府 | 脾胃、消化系统 |
| 太阴 | 眼睛、肾、内分泌 |

| 巨门 | 口腔、食道、皮肤 |
| 天相 | 面部、皮肤 |
| 天梁 | 脾胃、老年病 |
| 七杀 | 肺、大肠、外伤 |
| 破军 | 泌尿生殖、外伤 |

**煞星加重效应**：
| 煞星 | 加重方向 |
|------|---------|
| 擎羊 | 外伤、手术、发炎 |

### 7. 贵人与小人（命盘六合/六冲 + 吉星位置）【补全】

**数据来源**：命宫地支 + 年干 + 天魁天钺位置 + 六合六冲关系

**输出内容**：
- 贵人生肖（2个）
- 小人生肖（2个）
- 化解建议

**贵人推导规则**：

方法一：天乙贵人表（根据年干/日干）：
| 日干/年干 | 贵人地支 |
|----------|---------|
| 甲、戊 | 丑、未 |
| 乙、己 | 子、申 |
| 丙、丁 | 亥、酉 |
| 庚、辛 | 寅、午 |
| 壬、癸 | 卯、巳 |

贵人地支 → 对应生肖即为贵人生肖。

方法二：天魁天钺位置（紫微斗数贵人星）：
- 天魁所在宫位地支 → 对应生肖为阳贵人（男性贵人/长辈贵人）
- 天钺所在宫位地支 → 对应生肖为阴贵人（女性贵人/异性贵人）
- 两种方法取交集优先，无交集则合并取最多 2-3 个

**小人推导规则**：
- 命宫地支的六冲地支 → 对应生肖为小人
- 化忌星所在宫位地支 → 对应生肖为潜在阻碍

**六冲对照**：
| 地支 | 冲 |
|------|-----|
| 子 | 午 |
| 丑 | 未 |
| 寅 | 申 |
| 卯 | 酉 |
| 辰 | 戌 |
| 巳 | 亥 |

---

### 8. 重大决策时机（流年/流月择时）【补全】

**数据来源**：流年四化 + 流月天干地支 + 各宫位状态

**输出内容**：
- 跳槽/求职最佳月：官禄宫化禄或化权的流月
- 签约/合作最佳月：迁移宫或官禄宫化禄的流月
- 搬家/置业最佳月：田宅宫化禄的流月
- 表白/求婚最佳月：夫妻宫化禄或桃花星旺的流月
- 投资最佳月：财帛宫化禄的流月

**择时原则**：
1. 化禄飞入目标宫位的月份 → 最佳
2. 化权飞入目标宫位的月份 → 次佳（有掌控力）
3. 化忌飞入目标宫位的月份 → 回避
4. 多重吉化叠加月份 → 最优先推荐

**流月天干推算（年上起月法）**：
| 流年天干 | 正月（寅月）起始天干 |
|---------|-------------------|
| 甲、己 | 丙寅 |
| 乙、庚 | 戊寅 |
| 丙、辛 | 庚寅 |
| 丁、壬 | 壬寅 |
| 戊、癸 | 甲寅 |

从正月天干起，依次顺推：正月→二月→…→十二月，天干按甲乙丙丁…顺序轮转。每月天干确定后，查四化表得出该月四化飞入哪些宫位。

## 三、星级评分标准

所有"星级"评分统一使用 1-5 星制：

| 星级 | 判定条件 |
|------|---------|
| ★☆☆☆☆ (1) | 化忌落本宫 + 煞星集聚，严重受克 |
| ★★☆☆☆ (2) | 化忌落三方或主星落陷，偏弱 |
| ★★★☆☆ (3) | 无明显吉凶，中性平稳 |
| ★★★★☆ (4) | 化禄/化权/化科落本宫或三方，有吉星辅佐 |
| ★★★★★ (5) | 多重吉化叠加 + 主星庙旺 + 吉星拱照，极佳 |

## 四、多语言输出

本项目支持中英双语。JSON 中的文本字段按以下规则处理：
- `body`、`advice`、`detail` 等解读正文：根据 locale 输出对应语言
- `keywords`：中文输出中文词，英文输出英文词
- `soulmate`：中文"年长 · 属兔"，英文"Older · Rabbit sign"
- 方位：中文"东南"，英文"Southeast"
- 月份：中文"三月"，英文"March"

英文解读风格同样遵循"像朋友聊天"原则，避免过于学术化的翻译。
